## Gateway
Точка входа в систему сервисов. Принимает запросы на перебор MD5 хэша для поиска
слова(хэш и максимальная длина слова) после чего сразу отдаёт идентификатор запроса.
Считает количество комбинаций для максимальной длины, распределяет комбинации
по N worker'ам, ожидает от них ответа. После получения ответа изменяет состояние запроса(
готово, частично готово, ошибка, время вышло).
